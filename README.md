Password Generator

A simple and user-friendly Password Generator application built using Python Tkinter. This application allows users to generate strong and random passwords of any desired length and copy them directly to the clipboard for easy use.

📌 Project Overview

The Password Generator is a desktop GUI application designed to create secure passwords using a combination of uppercase letters, lowercase letters, numbers, and special characters. The project demonstrates the use of Python's random and string modules along with Tkinter GUI components.

✨ Features
Generate random passwords instantly.
User-defined password length.
Includes:Uppercase letters (A-Z), Lowercase letters (a-z), Numbers (0-9), Special characters (!, @, #, $, etc.)
Copy generated password to clipboard.
Input validation for incorrect entries.
Clean and attractive sky-blue interface.
Beginner-friendly implementation.
